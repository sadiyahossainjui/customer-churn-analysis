# Customer Churn Analysis

## Project Overview
This project analyzes customer churn behavior using Python, Pandas, and Tableau.

The goal was to identify factors affecting customer churn and visualize important business insights.

---

## Tools Used
- Python
- Pandas
- Google Colab
- Tableau
- Data Visualization

---

## Key Tasks
- Cleaned customer data
- Explored churn distribution
- Analyzed churn by:
  - Contract Type
  - Internet Service
  - Monthly Charges
  - Tenure
- Built an interactive Tableau dashboard

---

## Dashboard Preview

![Dashboard](dashboard.png)

---

## Key Insights
- Month-to-month customers churn the most
- Fiber optic users have higher churn
- Higher monthly charges are associated with more churn
- Long-term customers churn less
